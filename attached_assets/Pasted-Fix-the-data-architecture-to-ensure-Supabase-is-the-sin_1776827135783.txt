Fix the data architecture to ensure Supabase is the single source of truth, remove any local database dependency, implement automatic time awareness, and add a CGM connection status indicator (mocked).

--------------------------------------------------
🎯 GOAL
--------------------------------------------------

Ensure:

- Supabase is the ONLY database
- No local DB usage
- Google Sheets remains secondary sync layer
- Every log entry includes accurate time + timezone
- Add a visual CGM/BGM connection indicator (mocked)

--------------------------------------------------
🚨 1. REMOVE LOCAL DATABASE
--------------------------------------------------

Remove all usage of:

- local PostgreSQL
- SQLite
- file storage
- in-memory persistence

System must rely ONLY on Supabase.

--------------------------------------------------

User logs entry →
POST /api/log →
Insert into Supabase →
Return success

🧾 3. SERVER-SIDE LOG ENDPOINT
--------------------------------------------------

Create:

/src/app/api/log/route.ts

- validate input
- ensure timestamp

- return row

--------------------------------------------------

- timezone
