Task: Build a public brand guidelines website at /brand. This is an interactive single-page brand book that anyone (press, designers, partners) can visit at glev.app/brand to see the visual system, copy guidelines, and download brand assets. Match the existing app's dark aesthetic.

## File structure

Create these files:
- app/brand/page.tsx (the brand book page itself)
- app/brand/layout.tsx (optional — if you need different metadata than the app shell)
- components/brand/HexSwatch.tsx (click-to-copy color swatch component)
- components/brand/LogoCard.tsx (logo display with download button)
- components/brand/LogoPlayground.tsx (interactive GlevLogo with size/color/bg controls)
- components/brand/SectionNav.tsx (sticky anchor navigation)

Do NOT create new logo files. Import from the existing public/icon.svg and components/GlevLogo.tsx.

## Page architecture

Single long-scroll page, dark theme, max-width 1100px content column. Sticky top navigation with anchor links: Foundation · Color · Typography · Logo · Voice · Compliance.

Background: #09090B. Cards: #111117 with rgba(255,255,255,0.08) borders. Text primary: #FFFFFF. Use Inter font (already in app).
