STEP 2 — Add state:
const [mealTime, setMealTime] = useState<string>(""); // format: YYYY-MM-DDTHH:mm
const [mealTimeDirty, setMealTimeDirty] = useState(false);

onChange handler sets mealTime and flips mealTimeDirty to true. resetForm() resets both.

STEP 3 — Auto-populate from CGM fetch:
- When CGM fetch succeeds, the response includes a timestamp. If (!mealTimeDirty) setMealTime(toDatetimeLocal(cgmTimestamp)).
- Helper function:
  function toDatetimeLocal(iso: string): string {
    const d = new Date(iso);
    const pad = (n: number) => String(n).padStart(2, '0');
    return `${d.getFullYear()}-${pad(d.getMonth()+1)}-${pad(d.getDate())}T${pad(d.getHours())}:${pad(d.getMinutes())}`;
  }
- If the form is mounted with no CGM data yet, default mealTime to the current time (same helper called with new Date().toISOString()).
